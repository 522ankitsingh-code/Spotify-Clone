🎧 Spotify Clone (Frontend Project)

A responsive Spotify Web Player Clone built using HTML, CSS, and JavaScript.
This project replicates the basic layout and music player UI of Spotify including song cards, navigation bar, music controls, and playlists.

This project is part of my web development practice and portfolio as a B.Tech CSE student.

🚀 Features
🎵 Spotify-like UI design
▶️ Play / Pause music functionality
⏭️ Forward & Backward controls
🔀 Shuffle & Repeat buttons
📊 Music progress bar
🎶 Popular, Recommended & Recently Played sections
📱 Responsive layout
🔍 Search bar UI
📚 Library section
🛠️ Technologies Used
HTML5
CSS3
JavaScript
Font Awesome Icons
Google Fonts (Montserrat)
